# My Bucket List

A simple **C# WinForms application** to manage your bucket list dreams. You can **Add, Load, Update, and Delete** dreams with categories, all stored locally using **SQLite**.  

This app has a **fashionable and modern UI** with colored buttons and a styled DataGridView.  

## Features

- Add new dreams with categories  
- Load all dreams from the database  
- Update selected dream  
- Delete selected dream  
- Stylish interface with colored buttons and DataGridView  

## Requirements

- **Visual Studio 2022** (or later) with `.NET Desktop Development` workload  
- **.NET 6 or later**  
- **System.Data.SQLite** NuGet package
